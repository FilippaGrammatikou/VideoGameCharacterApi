## Overview

This folder contains the API controllers responsible for receiving HTTP requests and returning HTTP responses.

In this project, the controller layer stays thin and delegates application logic to the service layer.

---

## Current controller

### `VideoGameCharactersController`

Handles the CRUD endpoints for the video game character domain.

---

## Purpose

This folder represents the HTTP-facing layer of the project and reinforces the separation between endpoints, services, and data access.
